Types of Positions

In this section, we will focus on applying 5 different types of positions attribute in today's practice.

  • static

    A default option of positioning an element in HTML. Not effected by top, bottom, left, right. Static is always positioned according to the flow of the page.

  • relative

    An element with position: relative; is positioned relative to its normal position.



  • fixed

    Similar to static, but it is fix to the viewport, not the content of any element.



  • absolute

    Absolute is positioned based on the neaserst positioned ancester (HTML body if not found any).


  • sticky

    Elements with sticky as value is positioned based on the scroll position, it is pretty much the combination of relative and fixed.
